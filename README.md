# Yuliia Strategy Session — Лендинг для Kiro

Лендинг, що продає запис/участь у стратсесії з Танею за **149 грн**.

## Як використовувати

1. **Відкрити цю папку в Kiro:** File → Open Folder → `yuliia-strategy-session`
2. **Kiro автоматично завантажує** `.kiro/steering/` як контекст про проєкт.
3. **Дати Kiro команду:** «Прочитай docs/spec.md і збудуй лендинг.»
4. **Ітерувати:** переглядай diff-и, апрувай, проси правки.
5. **Деплой:** у самому кінці кажи «Налаштуй деплой на Vercel і поясни кроки мені».

## Структура

```
.kiro/steering/      ← контекст про проєкт (Kiro читає при кожній відповіді)
  product.md         ← що продаємо
  audience.md        ← хто наша аудиторія (Олена)
  brand.md           ← голос, тон, кольори, заборонені слова
  tech.md            ← стек технологій
  structure.md       ← як організована папка коду

docs/                ← довідкові матеріали
  spec.md            ← головний бриф на лендинг
  references/        ← скріншоти референс-сайтів (додай свої)
```

## Прийняті рішення

- **Фреймворк:** Astro 4 + Tailwind CSS
- **Платежі:** Wayforpay
- **Хостинг:** Vercel
- **Мова:** українська
- **Тариф для MVP:** одна сторінка (лендинг) + сторінка-дякую після оплати
