## Інтеграція Wayforpay

### Спрощений flow

```
[Кнопка на сайті]
    ↓
[POST /api/create-payment з фіксованими параметрами]
    ↓
[Сервер генерує підпис Wayforpay і повертає HTML-форму]
    ↓
[Прихована форма submit-иться автоматично → редирект на Wayforpay]
    ↓
[Wayforpay сторінка: користувач вводить ім'я, email, картку]
    ↓
[Оплата успішна → redirect на /thank-you]
    ↓
[Wayforpay серверний callback на /api/wayforpay-callback]
    ↓
[Сервер перевіряє підпис, надсилає email клієнту з Zoom + додатковою інфою про сесію]
```

### API endpoint: `src/pages/api/create-payment.ts`

```ts
import crypto from 'crypto';

export const prerender = false;

export async function POST({ request }) {
  const orderReference = `ORDER-${Date.now()}`;
  const orderDate = Math.floor(Date.now() / 1000);
  const amount = 149;
  const currency = 'UAH';
  const productName = ['Стратсесія 1 з 3 — позиціонування і оффер за 3 години'];
  const productCount = [1];
  const productPrice = [amount];

  const signatureString = [
    process.env.WAYFORPAY_MERCHANT_ACCOUNT,
    'yuliialaiko.com',
    orderReference,
    orderDate,
    amount,
    currency,
    ...productName,
    ...productCount,
    ...productPrice,
  ].join(';');

  const merchantSignature = crypto
    .createHmac('md5', process.env.WAYFORPAY_SECRET_KEY!)
    .update(signatureString)
    .digest('hex');

  return new Response(
    JSON.stringify({
      merchantAccount: process.env.WAYFORPAY_MERCHANT_ACCOUNT,
      merchantAuthType: 'SimpleSignature',
      merchantDomainName: 'yuliialaiko.com',
      orderReference,
      orderDate,
      amount,
      currency,
      productName,
      productCount,
      productPrice,
      merchantSignature,
      returnUrl: 'https://yuliialaiko.com/thank-you',
      serviceUrl: 'https://yuliialaiko.com/api/wayforpay-callback',
      language: 'UA',
    }),
    { headers: { 'Content-Type': 'application/json' } }
  );
}
```

### Webhook: `src/pages/api/wayforpay-callback.ts`

1. Приймає POST від Wayforpay
2. Перевіряє підпис
3. Отримує дані клієнта (email, ім'я) з payload Wayforpay
4. Записує в логи (для MVP — просто email Юлії або Google Sheet через webhook)
5. Надсилає email клієнту з Zoom-посиланням і додатковою інформацією про сесію
6. Повертає Wayforpay підтвердження прийняття callback

Для MVP можна обмежитись простим email через **Resend** або через **MailerLite API**.
